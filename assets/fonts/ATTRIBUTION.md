# Bundled fonts

## PT Sans Narrow

- Files: `pt_sans_narrow_regular.woff2` (weight 400), `pt_sans_narrow_bold.woff2` (weight 700).
- Subset: latin only (the woff2 latin subset served by Google Fonts, v19).
- Copyright (c) 2010, ParaType Ltd., with Reserved Font Names "PT Sans" and "ParaType".
- License: SIL Open Font License, Version 1.1. Full text in [OFL.txt](OFL.txt).
- Source: Google Fonts (https://fonts.google.com/specimen/PT+Sans+Narrow).

The OFL permits embedding and redistribution. The font is used here as the
condensed face for scene-object labels, declared via `@font-face` in
`src/style.css` and copied into `dist/assets/fonts/` by `build_github_pages.sh`.
