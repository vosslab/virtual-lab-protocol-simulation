Labels are hidden by default and revealed only when the student hovers or keyboard-focuses an object, keeping a dense scene visually quiet while preserving full discoverability. The reveal is pure CSS using :hover and :focus-within on .placement, with opacity and transform transitions for a soft fade. Accessibility is preserved because every placement carries title and aria-label attributes that screen readers and native tooltips can read regardless of label visibility.
