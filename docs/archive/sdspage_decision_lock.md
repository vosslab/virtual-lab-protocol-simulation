# SDS-PAGE spec-gap decision lock

This artifact is the single source of truth for the 20 spec gaps (SG1-SG20) enumerated in
the SDS-PAGE YAML implementation plan (`keen-finding-tarjan.md`, Milestone M0 / WP-0.2).
The plan's default recommendation is the starting answer for every SG; where the plan's
"Execution governance for the manager" -> "Allowed simplifications" block restates a
default, that exact phrasing is the locked answer. M1-M5 doers must treat the `decision`
column below as the authoring contract and must not re-derive a different answer from the
SG prose. Items the stepper cannot exercise today (`select`, branching, typed free text,
re-do loops) are recorded as prose-only with the deferral named in the follow-up column.
