# Off-canvas validation baseline

Report-only per-item off-canvas classification (WS-F, milestone M3). This baseline is generated by `tools/offcanvas_baseline.mjs` from the layout engine's report-only `offCanvasDiagnostics` stream at the canonical 16:9 frame (1920x1080).

The diagnostic only reports; it never fails or blocks a build. Each escaping item is graded against `scene_bounds`:

- `fully_off_canvas` (error class): the whole artwork box sits outside `scene_bounds` (the object is invisible).
- `partial_overflow` (warning): the box crosses a `scene_bounds` edge; the warning magnitude scales with the worst-axis overshoot in scene-percent (minor / moderate / severe tier).
