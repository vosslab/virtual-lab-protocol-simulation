# Pointer: durable scene layout baseline

The committed, hand-refreshed post-gate layout baseline lives at
[docs/SCENE_LAYOUT_BASELINE.md](../../SCENE_LAYOUT_BASELINE.md), not in this
`reports/` folder. That file is intentionally durable (refreshed by hand per
release); everything under `docs/active_plans/reports/` stays a regenerated or
point-in-time working artifact.

See also `docs/active_plans/reports/layout_diagnostics_baseline.md`, the
always-current, regenerated `severityDiagnostics` table that
`docs/SCENE_LAYOUT_BASELINE.md` snapshots from.
