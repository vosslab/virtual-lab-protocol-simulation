Anti-drift boundaries (must hold across the whole pipeline)

  • [CLOSED] Manifest schema is closed: no extras, no metadata, no open maps. New fields require a spec edit, not a YAML edit.
  • [CLOSED] Scene class list is closed and finite: template, composition, dense_clutter, instrument_heavy, zoom_detail. Authors pick, they do not invent.
  • [FORBIDDEN] No coordinates in the authoring layer. Authors name objects and a scene class; the CSS-native layout owns geometry.
  • [FORBIDDEN] Diagnostics and scorecard rules may not be modified to make a result pass. A failure is evidence, not a bug in the audit.
  • [GUARDRAIL] Bridge integrity: placement count cannot decrease after the layout/measure step. Lost objects fail the run.
Each red marker on the diagram corresponds to one or more of the boundaries above. The pipeline is intentionally one-way: lower stages may not write upward into earlier stages.
